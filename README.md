# Bridge

## Integrantes

* Manuela Carrasco
* Esteban Echeverri

## Introducción

Este es un programa en arduino para el manejo de un puente basculante(subida o bajada) por medio de una aplicación móvil, conectada por bluetooth, y sensores en el mismo puente.

## Software necesario

* Arduino IDE

## Hardware necesario

* Placa de Arduino Mega 2560
* Sensor de ondas
* Bluetooth HC-05
* Cables varios
* Protoboard
* Motor
